Sequence to Sequence (Послідовність у послідовність).
Для прогнозування часових рядів (як-от ціни на акції, курси валют або попит на товари) 
архітектура Sequence to Sequence (Many-to-Many) є дуже ефективною, 
оскільки вона дозволяє враховувати динаміку минулого для передбачення майбутнього на кілька кроків вперед.
