### Задача класифікації
    Маємо дуже розбалансований датасет, де категорія 0 має 67148 клієнтів, а категорія 1 - 5013.
    Після обробки датасету, rescalling/encoding, маємо на виході тренувальний датасет, тестувальний, 
    та маленьку, окрему частинку тренувального датасету для досліджень валідації.

    Було використано моделі XGBOOST та LGBOOST. 
    Окремо, для порівняння, було застосовано метод SMOTE разом з моделю XGBOOST.
    Метод небезпечний, оскільки дуже висока вірогідність перенавчання і точність в результаті нижча ніж без SMOTE.

    Послідовність файлів:
        - EDA - перший файл з препроцесінгом датасету - на виході отримуємо датасети, з якими потім працюємо.
        - lgb_kfold / xgb_kfold / Smote_xgb - файли, що містять моделі, безпосереднє навчання моделей.
          на виході зберігають навчені моделі у форматі .joblib
        - Model_analysis - до файлу підгружаються навчені моделі .joblib. І з використанням валідаційного датасету,
          проводиться дослідження і порівняння результатів.
