    Мета даного проєкту пройти життєвий цикл end-to-end продукту.  
    Для прикладу було вирішено працювати із задачею регресії для визначення вартості аланійської нерухомості.  
    Цикл передбачає:  
     - Самостійний збір даних (scraping / Jupyter Notebook)
     - Створення власної бази даних (DBeaver)
     - BI візуалізація (Tableau)
     - Навчання і валідація моделей (ML models / Jupyter Notebook)
     - Інтеграція моделі через API (FastApi)
     - Контейнеризація (Docker)  

     
### Самостійний збір даних (scraping)  
    Для збору даних зупинилась на сайті https://www.ataberkestate.com/turkey/alanya.  
    В них зручний сайт, де одразу, на зовнішній картці, можна знайти усі важливі ознаки.  
    Сайт має поверхневий antibot захист, тому просто за допомогою запиту requests витягти дані не вийшло.  
    Довелось застосувати webdriver з бібліотеки selenium.  
    Для парсингу HTML використовувалась бібліотека BeautifulSoup.  
    Було оброблено 270 сторінок сайту.  
    В результаті отримали 1556 об'єктів нерухомості, з ознаками у форматі:  
    
    '140 000 € | Комнат:2+1 | Площадь:100 m² | Район: Алания / Центр | До моря:550 м'  
    
    Після парсингу даних, чистки, вилучення дивних об'єктів, як, наприклад маленькі готелі на 10 - 14 кімнат,  
    отримали 1485 об'єктів у датасеті.  
    З міркування Feature engineering було додано ще дві ознаки, вартість квадратного метру, клас нерухомості.
