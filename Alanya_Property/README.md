# End-to-End Data Science Проєкт: Прогнозування вартості нерухомості (Аланія)

**Мета проєкту:** Реалізувати повний життєвий цикл розробки Data-продукту (End-to-End),  
від збору сирих даних до розгортання готової моделі в Production.

В якості доменної області обрано задачу регресії — визначення ринкової вартості об'єктів нерухомості в Аланії.

### Самостійний збір даних (scraping)  
    Джерелом даних виступив веб-ресурс Ataberk Estate.  
    Сайт має зручну структуру картки об'єкта із винесеними ключовими ознаками,  
    проте захищений від базових скриптів автоматизації (Anti-bot protection).
    
    Довелось застосувати webdriver з бібліотеки selenium.  
    Для парсингу HTML використовувалась бібліотека BeautifulSoup.  
    Було оброблено 270 сторінок сайту.  
    В результаті отримали 1556 об'єктів нерухомості, з ознаками у форматі:  
    
    '140 000 € | Комнат:2+1 | Площадь:100 m² | Район: Алания / Центр | До моря:550 м'  

    В процесі передоброки даних було виконано очищення датасету та вилучено нетипові аномальні об'єкти  
    (наприклад, комерційні міні-готелі на 10-14 кімнат), що скоротило вибірку до 1485 чистих записів. 
    Для покращення прогнозної сили моделей було згенеровано нові ознаки (Feature Engineering):  
    вартість квадратного метра та клас нерухомості (econom, comfort, lux).

### Створення власної бази даних (DBeaver)  

    Зібраний та очищений датасет було імпортовано до реляційної бази даних через клієнт DBeaver  
    для подальшого аналітичного менеджменту. 

    Workaround обмежень інструментів: Оскільки безкоштовна версія Tableau Public  
    має обмеження на пряме підключення до локальних баз даних, було реалізовано наступне:  
    SQL-запити (віконні функції, агрегації) виконувалися безпосередньо в базі SQLite,  
    після чого агреговані результати експортувалися в .csv файли для подальшого завантаження в Tableau.

### BI візуалізація (Tableau)
    Результатом попереднього пункту став даний дашборд:  

🔗 [Дивитися інвестиційний дашборд на Tableau Public](https://public.tableau.com/views/AlanyaRealEstateInvestmentDashboard/Dashboard1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

**Ключові аналітичні блоки дашборду:**

    1. Price Deviation (Scatter Plot): Візуалізує хмару пропозицій ринку та підсвічує кольоровим градієнтом
    (від зеленого до червоного) недооцінені об'єкти, чия вартість нижча за ринковий медіанний зріз.
     Доступна інтерактивна фільтрація за районами та типом планування (кількістю кімнат).  
    
    3. Budget Segment (Heatmap): Матриця розподілу пропозицій, яка класифікує нерухомість за чотирма ціновими бакетами
     та демонструє щільність ринку (кількість доступних об'єктів) у розрізі кожного району Аланії.
    
    5. Price per Square Meter (Bar Chart): Дає повну довідку щодо вартості квадратного метра (мінімальна, середня, максимальна ціна),
     відображає загальний обсяг пропозицій та аналізує середню географічну відстань до моря по кожній локації.
