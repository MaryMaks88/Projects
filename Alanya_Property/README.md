# End-to-End Data Science Проєкт: Прогнозування вартості нерухомості (Аланія)

**Мета проєкту:** Реалізувати повний життєвий цикл розробки Data-продукту (End-to-End),  
від збору сирих даних до розгортання готової моделі в Production.

В якості доменної області обрано задачу регресії — визначення ринкової вартості об'єктів нерухомості в Аланії.

### Самостійний збір даних
Джерелом даних виступив веб-ресурс Ataberk Estate.  
Сайт має зручну структуру картки об'єкта із винесеними ключовими ознаками,  
проте захищений від базових скриптів автоматизації (Anti-bot protection).

Довелось застосувати `webdriver` з бібліотеки `selenium`.  
Для парсингу HTML використовувалась бібліотека `BeautifulSoup`.  
Було оброблено 270 сторінок сайту.  
В результаті отримали 1556 об'єктів нерухомості, з ознаками у форматі:  

`'140 000 € | Комнат:2+1 | Площадь:100 m² | Район: Алания / Центр | До моря:550 м'`  

В процесі передоброки даних було виконано очищення датасету та вилучено нетипові аномальні об'єкти  
(наприклад, комерційні міні-готелі на 10-14 кімнат), що скоротило вибірку до 1485 чистих записів. 
Для покращення прогнозної сили моделей було згенеровано нові ознаки (Feature Engineering):  
вартість квадратного метра та клас нерухомості (econom, comfort, lux).

### Створення власної бази даних (DBeaver)  

Зібраний та очищений датасет було імпортовано до реляційної бази даних через клієнт `DBeaver`  
для подальшого аналітичного менеджменту. 

Workaround обмежень інструментів: Оскільки безкоштовна версія `Tableau Public`  
має обмеження на пряме підключення до локальних баз даних, було реалізовано наступне:  
SQL-запити (віконні функції, агрегації) виконувалися безпосередньо в базі `SQLite`,  
після чого агреговані результати експортувалися в .csv файли для подальшого завантаження в `Tableau`.

### BI візуалізація (Tableau)
Результатом попереднього пункту став даний дашборд:  

🔗 [Дивитися інвестиційний дашборд на Tableau Public](https://public.tableau.com/views/AlanyaRealEstateInvestmentDashboard/Dashboard1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

**Ключові аналітичні блоки дашборду:**

1. **Price Deviation (Scatter Plot)**: Візуалізує хмару пропозицій ринку та підсвічує кольоровим градієнтом
(від зеленого до червоного) недооцінені об'єкти, чия вартість нижча за ринковий медіанний зріз.
Доступна інтерактивна фільтрація за районами та типом планування (кількістю кімнат).  

2. **Budget Segment (Heatmap)**: Матриця розподілу пропозицій, яка класифікує нерухомість за чотирма ціновими бакетами
та демонструє щільність ринку (кількість доступних об'єктів) у розрізі кожного району Аланії.

3. **Price per Square Meter (Bar Chart)**: Дає повну довідку щодо вартості квадратного метра (мінімальна, середня, максимальна ціна),
відображає загальний обсяг пропозицій та аналізує середню географічну відстань до моря по кожній локації.


### Навчання та валідація моделей

Для побудови надійної системи прогнозування вартості нерухомості було реалізовано ітеративний підхід:  
від найпростішої базової моделі (Baseline) до сучасних ансамблів градієнтного бустингу.

У межах дослідження було протестовано 6 різних архітектур:

1. Ridge Regression (Лінійна модель з L2-регуляризацією)
2. Decision Tree Regressor (Поодиноке дерево рішень)
3. Random Forest Regressor (Бенчмарк на основі беггінгу)
4. XGBoost (Градієнтний бустинг)
5. LightGBM (Гістограмний градієнтний бустинг)
6. CatBoost (Бустинг на основі симетричних дерев)


📊 **Попередня обробка даних та інженерія ознак**    

***Мультиколінеарність ознак:***   
Етап розвідувального аналізу даних (EDA) виявив сильну лінійну кореляцію ($r = 0.90$)  
між ознаками rooms (кількість кімнат) та area_sq_m (площа об'єкта).  
Враховуючи обмежену кількість ознак у датасеті, видалення однієї з них призвело б до втрати важливого контексту.  
Рішення: Як базову лінійну модель було обрано саме Ridge Regression.   
Штрафування великих ваг за допомогою L2-регуляризації дозволило компенсувати негативний вплив мультиколінеарності без втрати ємності даних.

***Трансформація цільової змінної (Target Transformation):***
Розподіл цін на нерухомість в Аланії має класичний правосторонній скос  
(наявність невеликої кількості елітних вілл та пентхаусів).  
Для стабілізації процесу навчання та вирівнювання відносної   
помилки прогнозування було застосовано логарифмування таргету ($\log1p$).  
Усі фінальні метрики розраховані після зворотного перетворення прогнозованих значень через експоненту ($\text{expm1}$).

***Підготовка числових та категоріальних даних (для Ridge, Trees, XGBoost, LightGBM):***
Для числових ознак застосовано масштабування StandardScaler.
Для категоріальної ознаки district, rooms, class,  використано кодування з бібліотеки category_encoders.  

***Особливість підготовки для CatBoost:***
Для моделі CatBoost числові ознаки не масштабувалися,  
а категоріальна колонка district передавалася у сирому текстовому вигляді через параметр cat_features.   
Алгоритм використав свій вбудований та більш ефективний механізм Ordered Target Encoding,  
що повністю захистило модель від витоку даних (Data Leakage).

Метрики якості (MAE пораховано в реальних євро після зворотного перетворення таргету):

| Модель | MAE (Середня помилка) | R2 Score (Точність) | Тип |

| Ridge Regression | 23 304.17 € | 0.9235 | Baseline |
| Decision Tree | 18 809.19 € | 0.9300 | Single Tree |

| CatBoost | 3 670.70 € | 0.9964 | Boosting |


👑 **Аналіз моделі-переможця (CatBoost)**  
Абсолютним лідером за всіма метриками стала модель CatBoost,   
продемонструвавши високу точність прогнозування із середньою помилкою всього 3 670.70 €   
та коефіцієнтом детермінації $R^2 = 0.9964$.  
Аналіз кривої навчання (Learning Curve) підтверджує, що модель стабілізувалася після 400-ї ітерації.   
Синхронний рух метрик на тренувальному (learn) та валідаційному (test) наборах свідчить про повну відсутність перенавчання (Overfitting).

**Діагностика прогнозів моделі:**  
***Residuals Plot (Графік залишків):***   
Демонструє високу концентрацію помилок біля нуля в масовому бюджетному сегменті нерухомості (до 300 000 €).   
Зі збільшенням вартості об'єктів спостерігається природне для ринку розширення дисперсії залишків,  
спричинене унікальними ціновими факторами преміум-сегменту.
